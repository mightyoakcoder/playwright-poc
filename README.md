# Playwright Testing for Mobile Adapter

### Getting Started
- clone repo + run npm install

### Ways to Run Tests
- you can run these commands in terminal:
  - npm run test:api
  - npx playwright test
  - npx playwright tests/you-file.test.ts
  - npx playwright test tests/you-dir/
